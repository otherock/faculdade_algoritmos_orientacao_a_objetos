# Algoritmos e Programação: Orientação a Objetos (C++) 💻

Repositório destinado aos materiais, exercícios e trabalhos da atividade acadêmica de **Algoritmos e Programação: Orientação a Objetos** na **UNISINOS**.

---

## 📚 O que iremos aprender:
O curso inicia com uma revisão de lógica em C e avança para os conceitos fundamentais de POO e Estruturas de Dados.

### Principais Tópicos:
* **Revisão C:** Funções, Vetores, Strings, Ponteiros e Alocação Dinâmica.
* **Fundamentos de POO:** Classes, Objetos, Atributos, Métodos e Construtores.
* **Aprofundamento POO:** Herança, Polimorfismo, Sobrecarga/Sobrescrita e Tratamento de Exceções. 
* **Estruturas de Dados:** Pilhas, Filas e Listas Encadeadas.
